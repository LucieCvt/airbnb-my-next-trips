Utilisé pour `padding`, `margin`, `gap`, etc. Dérivé de size.

| -------- | ---- |
| `$0`     | 0    |
| `$0.25`  | 0.5  |
| `$0.5`   | 1    |
| `$0.75`  | 1.5  |

| `$2.5`   | 10   |
| `$3`     | 13   |
| `$3.5`   | 16   |
| `$4`     | 18   |
| `$true`  | 18   |

| `$5`     | 24   |

| `$8`     | 46   |
| `$9`     | 53   |
